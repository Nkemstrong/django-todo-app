## ⚙ Installation

Clone the repository

```bash
git clone https://github.com/Nkemstrong/django-todo-app.git
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run migrations

```bash
python manage.py migrate
```

Run the server

```bash
python manage.py runserver
```
